# simple-django-project
## Installation

### Prerequisites

#### 1. Install Django
```bash
pip install django
```
#### 2.Run the server on port 8080
```bash
cd proj prev/
python manage.py runserver 8080

# your server is up on port 8080
```
Try opening [http://localhost:8080](http://localhost:8080) in the browser.
Now you are good to go.
